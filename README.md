# Job Queue System 🚀

A comprehensive job queue system with Discord bot hosting, website monitoring, and real-time chat.

## ✨ Features

- 📦 **API Management**: Create and manage multiple APIs with custom settings
- 🤖 **Discord Bot Hosting**: Host and manage Discord bots directly from the web
- 📡 **Website Monitoring**: Monitor website uptime and receive alerts
- 💬 **Real-time Chat**: Built-in chat system using WebSocket
- 👥 **User Management**: Role-based access control (Owner, Admin, Member)
- 🔒 **Private APIs**: IP whitelisting and private mode
- 📊 **Dashboard**: Real-time statistics and monitoring

## 🚀 Quick Start

### Prerequisites

- Node.js >= 18.0.0
- Python >= 3.7 (for Discord bots)
